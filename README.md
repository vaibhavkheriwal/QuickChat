# Quick Chat

- Console-based chat application in C++.
- Provides a shared chat area for two or more users.
- Supports sending and receiving messages through an online endpoint.
- Displays message timestamps and user information.
- Uses JSON for message storage and transfer.
- Implements HTTP communication using libcurl.
- Parses and generates JSON data using nlohmann/json.
- Demonstrates networking, API communication, and console application development.
- User identifiers (`a`, `b`) are examples and can be changed as needed.
- Replace `YOUR_DATABASE_ENDPOINT` with your own server or database endpoint before running the application.

## Technologies Used

- C++
- libcurl
- nlohmann/json
- Firebase Database
